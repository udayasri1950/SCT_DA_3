# SCT_DA_3 - Interactive Visualization

## Objective
Create an interactive dashboard using Tableau to analyze HR Employee Attrition data.

## Tools Used
- Tableau Public
- HR Employee Attrition Dataset

## Dashboard Insights
- Research & Development department has the highest employee attrition.
- Sales department has the second-highest attrition.
- Human Resources department has the lowest attrition.
- Job satisfaction influences employee retention.
